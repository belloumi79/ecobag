- **Zo.space** : https://belloumi.zo.space/

## 🚀 Déploiement Vercel

### Méthode 1 : Vercel CLI (rapide)

```bash
npm i -g vercel
vercel --prod
```

### Méthode 2 : GitHub + Vercel (recommandé)

1. Poussez ce code sur GitHub
2. Connectez-vous sur https://vercel.com avec GitHub
3. Importez le dépôt
4. Vercel détecte automatiquement Next.js
5. Cliquez **Deploy**

### Configuration Vercel

| Paramètre | Valeur |
|-----------|--------|
| Framework | Next.js |
| Build Command | `next build` |
| Output Directory | `dist` |

## 🛠️ Développement local

```bash
# Installation
npm install

# Développement
npm run dev

# Build
npm run build

# Production
npm start
```
